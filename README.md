# AilyBlockly
灵感来自Github copilot  


## 测试&&打包  

**库安装**
```
npm i
cd electron
npm i
```  

**浏览器运行**
```
npm run start
```
**浏览器打包**
```
npm run build
```
打包后生成路径为dist\aily-blockly\browser

**electron运行**
```
npm run electron
```

**electron打包(github action用)**
```
npm run electron:build
```
打包后生成路径为dist\aily-blockly\win-unpacked

**electron打包(本地用，设置了代理，避免资源拉取失败)**
```
npm run electron:build-local
```
